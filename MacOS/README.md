# Codex Account Switcher

Локальный переключатель аккаунтов Codex для общего Mac.

## Как настроить

Иконка приложения хранится в `Assets/AppIcon-1024.png`; сборочный скрипт делает из неё `.icns`.

1. В приложении введите имя аккаунта, например `my`.
2. Нажмите `Войти`.
3. Codex откроется без старой сессии. Войдите в нужный аккаунт.
4. Переключатель сам заметит успешный вход, сохранит профиль и снова откроет Codex.
5. Повторите то же самое для второго аккаунта, например `brother`.
6. После этого выбирайте профиль и нажимайте `Переключиться`.

CLI-запасной вариант для уже активного аккаунта:

```bash
./scripts/codex-account-switcher.sh save my
```

В интерфейсе можно:

- сохранять текущую сессию Codex в новый профиль;
- импортировать профиль, просто закинув его папку в `~/.codex-account-switcher/profiles`;
- переключаться между профилями;
- видеть последний сохранённый снимок лимитов аккаунта;
- редактировать название профиля;
- удалять ненужные профили;
- открыть папку с локальными профилями.

## Что хранится

Профили лежат локально в `~/.codex-account-switcher/profiles`.

Для ручного импорта можно положить в эту папку профиль в одном из форматов:

- `Имя профиля/auth.json`;
- `Имя профиля/.codex/auth.json`;
- `Имя профиля/Application Support/Codex/...`;
- `Имя профиля/Library/Application Support/Codex/...`;
- `Имя профиля/Codex/...`;
- `Имя профиля/Cookies`, `Local Storage`, `Session Storage` и другие файлы сессии прямо внутри папки.

Приложение сохраняет:

- браузерную сессию Codex/ChatGPT из `~/Library/Application Support/Codex`;
- CLI-авторизацию `~/.codex/auth.json`, если она есть.

Пароли не запрашиваются и никуда не отправляются. Сохранённые cookies/token-файлы являются доступом к аккаунту, поэтому папка профилей создаётся с ограниченными правами.

Перед переключением текущий активный профиль обновляется, чтобы сохранить свежие cookies и токены.

Лимиты берутся из последних `rate_limits`, которые Codex пишет в `~/.codex/sessions`. Это снимок в процентах: приложение показывает оставшийся процент, время сброса и тип плана, если Codex уже записал эти данные.

## Команды

```bash
./scripts/codex-account-switcher.sh run
./scripts/codex-account-switcher.sh save <profile>
./scripts/codex-account-switcher.sh switch <profile>
./scripts/codex-account-switcher.sh list
./scripts/codex-account-switcher.sh status
```

Командный скрипт оставлен как запасной вариант для `auth.json`. Для реального входа ChatGPT/Codex используйте приложение с интерфейсом.
